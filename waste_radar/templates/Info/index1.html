GEOTAGGING

Waste Collection is in a relative state of infancy here in the Philippines. Lacking an efficient data structure of may contribute to its growth. In this project study, the proponents looked into the three collection points within a local barangay in Davao City. Predicting the number of bins as the measure for a volume of waste in a location point is important in providing an efficient solid waste collection service. This insight is all about how many bins will be allocated. The proponents utilized Geotagging for the identifying the coordinates of the location. The approach used is Simple Linear Regression, where the relationship between the number of filled bins and date (time) are observed. The result of the equation can be used to predict the bins quickly and making data-driven decisions seamlessly. The result proves that the project is indeed applicable and be able to contribute in the context of solid waste management.
